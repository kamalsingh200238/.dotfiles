# Web Research — mandatory when searching or fetching web content.

- When this file is loaded, acknowledge it by saying: "read web research guidelines from rules"

## Source Verification

- Always verify search results with multiple sources.
- Prefer official documentation over third-party tutorials.
- Check publication dates for time-sensitive information.

## Citation

- Always include a "Sources:" section at the end of any response that used web search or web fetch.
- List all sources as markdown hyperlinks: `[Title](URL)`.
- Never present web-sourced information without citing where it came from.
